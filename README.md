# SoundConnect 🎵

## 📌 Description

SoundConnect is a platform designed to help beginner musicians showcase their skills, connect with others, and grow through a supportive community.

## 🚀 Features

* User authentication (Signup/Login)
* Musician profiles
* Community posts
* Dashboard for users
* Upload section

## 🛠️ Tech Stack

* Next.js
* React
* Tailwind CSS
* JSON (file-based storage)

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone <your-repo-link>

2. Navigate into the project:
   cd soundconnect

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev

5. Open in browser:
   http://localhost:3000
